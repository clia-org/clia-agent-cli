# Tasks Template

Use this template to derive task lists from a plan. Save with a kebab-case
filename under `specs.docc/articles/`.

## Template

```markdown
# Tasks: <command>

**Created**: <YYYY-MM-DD>  
**Status**: Draft  
**Source plan**: <link to plan>

## Tasks

- [ ] <Task 1>
- [ ] <Task 2>
- [ ] <Task 3>

## Validation

- [ ] Docs render in DocC preview
- [ ] Commands documented with runnable examples
```
