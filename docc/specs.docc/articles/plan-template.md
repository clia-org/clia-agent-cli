# Plan Template

Use this template for the implementation plan that follows a spec. Save with a
kebab-case filename under `specs.docc/articles/`.

## Template

```markdown
# Implementation plan: <command>

**Created**: <YYYY-MM-DD>  
**Status**: Draft  
**Source spec**: <link to spec>

## Architecture

- <High-level approach>
- <Key dependencies or modules>
- <DocC bundle location>

## Execution steps

1. <Step 1>
2. <Step 2>
3. <Step 3>

## Risks and mitigations

- <Risk>: <Mitigation>
```
