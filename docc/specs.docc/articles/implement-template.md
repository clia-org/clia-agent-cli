# Implement Template

Use this template to capture the implementation log after tasks are executed.
Save with a kebab-case filename under `specs.docc/articles/`.

## Template

```markdown
# Implementation log: <command>

**Created**: <YYYY-MM-DD>  
**Status**: Draft  
**Source tasks**: <link to tasks>

## Changes

- <Change 1>
- <Change 2>

## Notes

- <Decisions or caveats>

## Verification

- <Tests or previews run>
```
