## Summary

Adopted two clear names so docs read like product, not plumbing:

- A*Triads (0.3) — Agent/Agency/Agenda triads with typed `notes` arrays and a
  unified `entries` list. The A* nod signals we optimize paths with good
  heuristics: predictable structure → better automation.
- S‑Type Contribution System — typed matrix for how contribution roles interact
  (synergy, stability, strain) with deterministic scoring and a CLI.

## Guidance

- Use the names in titles and prose. Keep slugs/file paths kebab‑case ASCII
  (for example, `triads-0-3-update.md`, `s-types.docc`).
- When linking across bundles, prefer explicit HTTP links over `<doc:…>`.
  DocC identifiers don’t cross bundle boundaries.
- Add a short “Naming” note at the top of overview pages to anchor newcomers.
